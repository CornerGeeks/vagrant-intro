# Notes

- This folder will be mounted in the vagrant box
- Once Vagrant box is running, ssh into the box with
  vagrant ssh
- Go into this folder:
  cd slides
- Clone a slides repo e.g. 
  git clone https://github.com/markdalgleish/presentation-tabs.git
- Go into the repo folder e.g. 
  cd presentation-tabs
- Install dependencies e.g. 
  npm install && bower install
  (noticed that npm install fails sometimes, just rerun and it will work)
- Serve the slides to be e.g. 
  gulp serve
